# NFL Draft Efficiency Analysis

## Project Goal
Evaluate whether drafting well correlates with long-term NFL success in the rookie wage scale era (2011–2023).

## Questions
- Does draft performance predict wins?
- How important are late-round picks?
- Do strong drafting teams sustain success?

## Structure
data/ – Raw and processed datasets  
notebooks/ – Jupyter notebooks  
outputs/ – Visualizations and final report
